Kavya Kanaja:

Kavya Kanaja is a modern Android application for exploring and preserving classical Kannada poetry through 
a clean and interactive digital platform. The app allows users to read poems, explore poet information, save
favourite poems, and listen to poems using text-to-speech functionality.

Features:
>Explore Kannada Poems,
>Poet Information,
>Favourite Poems,
>Modern UI with Jetpack Compose,
>Smooth Multi-Screen Navigation.

Technologies Used
>Kotlin,
>Jetpack Compose,
>MVVM Architecture,
>Room Database,
>Navigation Compose,
>Kotlin Coroutines,
>StateFlow,
>Android Studio,
>Project Objective.

The objective of this project is to digitally preserve Kannada literary heritage and improve accessibility to
classical Kannada poetry through a modern Android application.
